# Snake AI Duel

Classic terminal Snake game built with Python + curses, featuring:
- Player-controlled snake (WASD / arrow keys)
- Simple greedy AI opponent that chases the food
- Split-screen duel mode
- Score tracking + game over

Fun, clean demo project in exactly 9 commits.

```bash
python main.py
